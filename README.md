技术文档：


卡牌：
    使用CardData类来表示卡牌数据，包括图标、费用、冷却时间等。
    使用CardInteraction类来处理卡牌交互，包括选中、取消选中、点击等。
    使用CardDeploy类来处理卡牌部署，包括部署、取消部署等。

    创建CardData来调数据

    拓展：创建功能脚本，给CardData加个类型，给CardDeploy加个判断（选路径变选范围），根据部署状态来实现新功能（例如：在部署时触发）；
    或者自定义实现条件自动运行。

单位：
    使用UnitAttr类来表示单位数据，包括图标、费用、冷却时间等。
    使用UnitInteraction类来处理单位交互，包括选中、取消选中、点击等。
    使用UnitDeploy类来处理单位部署，包括部署、取消部署等。

    通过挂载UnitAttr来调数据

    拓展：直接添加功能脚本，由UnitAI来调用。或者自定义实现条件自动运行。
